#### 3. 用Newton法导出计算机上求$A^{\frac1x}$($A>0$,n为非零整数)和$\frac1A$的算法（即只用加、减、乘三种运算的算法），并实际计算下列各值：
（1）$\sqrt[3]{2}$；

（2）$\frac1{\sqrt[5]{9}}$；

（3）$\frac17$；

（4）$\frac1{11}$；

解：设$f(x)=x^n-A$,$f'(x)=nx^{n-1}$，迭代公式为$$x_{k+1}=x_{k}-\frac{x^n_k-A}{nx^{n-1}_k}=\frac{n-1}{n}x_k+\frac{A}{nx^{n-1}_k};$$
